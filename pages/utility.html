Variables

{% include br-branding.html %} makes heavy use of SASS variables. These variables can be found in /src/common/_vars.scss. As part of the build process, a variable override file is also created at /overrices.scss. The overrides file is automatically included as part of the build process and any variables defined in the override file will take precedence over the default value. This allows you to easily modify small parts of the {% include br-branding.html %} framework without having to modify core code.

For a complete list of all variables that you can override, consult the /src/common/_vars.scss file.

Mixins

{% include br-branding.html %} includes several pre-defined mixins. These mixins are located in /src/common/mixins. They are automatically loaded during the build process and can be used elsewhere in the SASS files (including in your custom CSS as long as you load the init.scss file first).

Font mixins

The most commonly used mixins are the font mixins, which standardize the use of fonts throughout Bedrock. There are five font mixins defined:

  • font_sans
  • font_sans_light
  • font_serif
  • font_mono
  • font_glyph
