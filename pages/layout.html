Containers

{% include br-branding.html %} has a basic container system to aid in layout. There are two container types:

  • Full-width: Container spans the entire available width of the parent element
  • Fixed-width: Container spans the entire available width of the parent element up to a max width as defined by $br_width_fixed
Note that the examples below have an additional background color added to make them more visible. In practice, the containers have no background color.

Grid

{% include br-branding.html %} includes a basic grid. The grid uses twelve columns, meaning that the column widths in any row of the grid should add up to 12. To create a grid, create a .br-row element and nest one or more .br-col-{n} elements, where {n} is the number of columns you want to span.

The grid system is reponsive and will collapse to a single column display when the viewport drops beneath the mobile width defined in the $br_width_mobile variable.
